# sql-python-data-automation (SQL, Python y automatización de reportes)

# SQL + Python Data Automation Project

Proyecto básico de análisis de datos orientado a demostrar habilidades en SQL, Python y automatización de reportes.

---

## Objetivo

- Practicar consultas SQL
- Automatizar extracción y análisis de datos
- Generar reportes simples a partir de datos

---

## Tecnologías

- SQL (SQLite)
- Python (Pandas)

---

## Estructura

- `data/`: datos en formato CSV
- `sql/`: consultas SQL
- `python/`: scripts de carga y reporte

---

## Proceso

1. Leer datos desde CSV
2. Cargar datos a base de datos SQLite
3. Ejecutar consultas SQL
4. Generar reporte automático en CSV

---

## Análisis realizados

- Total de dinero por usuario
- Cantidad de transacciones
- Ranking de usuarios por gasto

---

## Automatización

Script en Python que ejecuta consultas SQL y genera un reporte automáticamente.

---

## Cómo usar

1. Ejecutar ETL: python python/etl.py
2. Generar reporte: python python/report.py
